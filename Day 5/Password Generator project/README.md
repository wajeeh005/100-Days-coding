# Write a code of which create a strong Password.


1. Eazy Level - Order not randomised:
    #### Example 
        4 letter, 2 symbol, 2 number 
        password = JduE&!91

2.  Hard Level - Order of characters randomised:
    #### Example 
        4 letter, 2 symbol, 2 number 
        password = u1&Ed9!J

### Hint:
You can use random.shuffle() and random.choices()
